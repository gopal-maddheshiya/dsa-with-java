# Recursion

This package covers recursive problem-solving techniques and patterns.

## Topics Covered

- **Recursion Basics** — Understanding how recursive calls work, base cases, and recursive cases
- **Mathematical Problems** — Factorial, Fibonacci, and other mathematical recursions
- **Divide and Conquer** — Breaking problems into smaller subproblems
- **Backtracking Problems** — Finding solutions through trial and elimination
- **String Manipulation** — Removing duplicates and other string operations using recursion

## Practice Problems Included

- Recursion fundamentals
- Binary strings problem
- Tilling problem (tiling a 2×n board with 1×2 tiles)
- Friends pairing problem (n friends pairing combinations)
- Remove duplicates from strings using recursion
- Divide and conquer approaches

## Key Concepts

1. **Base Case** — Condition to stop recursion
2. **Recursive Case** — Problem decomposition and recursive calls
3. **Stack Trace** — Understanding the call stack
4. **Time Complexity** — Often O(2ⁿ) or O(n!) for exponential recursion
5. **Space Complexity** — O(n) for recursion depth

## How to Use

1. Trace through recursive calls manually to understand the flow
2. Identify base cases and recursive cases
3. Understand memoization for optimization (covered in advanced topics)
4. Practice drawing recursion trees

**Important:** Recursion can be memory-intensive. Consider iterative approaches for large inputs.

**Next Steps:** Learn dynamic programming and memoization to optimize recursive solutions.
