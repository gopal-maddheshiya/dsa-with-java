# Searching Algorithms

This package covers fundamental searching techniques used to find elements in data structures.

## Topics Covered

- **Linear Search** — Sequential search through elements one by one
- **Binary Search** — Efficient search algorithm for sorted data (O(log n) complexity)

## Practice Problems Included

- Linear search implementation
- Binary search implementation
- Variations and optimizations of binary search

## Complexity Analysis

- **Linear Search:** Time O(n), Space O(1)
- **Binary Search:** Time O(log n), Space O(1)

## How to Use

Understand when to use each searching algorithm based on your data structure and whether it's sorted. Binary search is more efficient but requires sorted data.

**Key Takeaway:** Choose the right searching algorithm based on your data characteristics. For unsorted data, use linear search. For sorted data, binary search is optimal.

**Next Steps:** Combine searching with sorting for complete problem-solving strategies.
