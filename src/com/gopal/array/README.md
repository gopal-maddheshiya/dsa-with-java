# Arrays

This package covers array-related data structures, operations, and problem-solving techniques.

## Topics Covered

- **Array Basics** — Declaration, initialization, and accessing array elements
- **Multi-dimensional Arrays** — 2D arrays and jagged arrays
- **Array Operations** — Reversing arrays, finding largest/smallest elements
- **Sub-arrays & Subarrays** — Understanding subarrays and their properties
- **Maximum Subarray Problems** — Kadane's algorithm, prefix sum approach
- **Searching in Arrays** — Finding specific elements in arrays
- **2D Array Problems** — Searching in 2D matrices and finding targets

## Practice Problems Included

- Finding the largest element in an array
- Reversing an array
- Finding pairs that sum to a target
- Maximum subarray sum (Kadane's algorithm)
- Maximum subarray sum (Prefix sum method)
- Maximum in a sub-array of size k
- Trapping rainwater (2D matrix variant)
- Buy and sell stocks problem
- Finding target in 2D matrix
- Runtime value updates in arrays

## How to Use

Study each file to understand different array manipulation techniques. Practice implementing solutions for array-based problems.

**Key Concepts:** Master arrays before moving to more advanced data structures like linked lists and trees.
