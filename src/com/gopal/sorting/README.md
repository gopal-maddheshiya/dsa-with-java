# Sorting Algorithms

This package implements various sorting algorithms with different time and space complexities.

## Topics Covered

- **Bubble Sort** — Simple sorting algorithm with O(n²) complexity
- **Selection Sort** — Finds minimum element and places it in correct position
- **Insertion Sort** — Builds sorted array by inserting elements one by one
- **Merge Sort** — Divide-and-conquer algorithm using recursion
- **Quick Sort** — Efficient divide-and-conquer algorithm with average O(n log n)
- **Counting Sort** — Non-comparative sorting for integers with limited range
- **Binary Search** — Included for searching in sorted data

## Complexity Comparison

| Algorithm | Best | Average | Worst | Space |
|-----------|------|---------|-------|-------|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) |
| Counting Sort | O(n+k) | O(n+k) | O(n+k) | O(k) |
