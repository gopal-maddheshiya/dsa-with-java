# Loops

This subdirectory contains fundamental loop practice problems and implementations.

## Topics Covered

- **For Loops** — Traditional for loops with counters
- **While Loops** — Conditional loops
- **Loop Patterns** — Number patterns, reverse numbers, and sequences

## Practice Problems Included

- Print 1 to N
- Print N in reverse
- Sum of first N numbers
- Square patterns using loops
- Square pattern printing

## How to Use

Master loops as they are essential for:
- Iterating through data structures
- Implementing algorithms
- Creating patterns and sequences

**Key Concept:** Loops are fundamental to programming. Practice until you can write loops without thinking about the syntax.
