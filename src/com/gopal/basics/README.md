# Java Basics

This package covers fundamental Java programming concepts and is the foundation for learning Data Structures and Algorithms.

## Topics Covered

- **Variables & Data Types** — Understanding primitive types, variable declarations, and type conversion
- **Operators** — Arithmetic, logical, bitwise, and comparison operators
- **Control Flow** — If-else statements, switch cases, and conditional logic
- **Loops** — For loops, while loops, do-while loops, and nested loops (in `loops/` subdirectory)
- **Functions & Methods** — Function declaration, function overloading, and return types
- **Input/Output** — Reading user input and printing output to console
- **Mathematical Concepts** — Factorial, prime numbers, palindromes, binary conversions
- **Bit Manipulation** — Working with binary representations and bitwise operations
- **Object-Oriented Basics** — Constructors, call by value, and basic class concepts
- **String Operations** — String manipulation, compression, and palindrome checking

## Practice Problems Included

- Area of a circle
- Average of numbers
- Prime number checking
- Palindrome verification
- Pattern printing
- Binary and decimal conversions
- Binomial coefficients
- Trapping rainwater problem

## How to Use

Each file demonstrates a specific concept with implementations. Review the code to understand how to apply these fundamental concepts in Java.

**Next Steps:** After mastering Java basics, move on to Arrays, Strings, and Searching algorithms for DSA practice.
